How It Works

This extension monitors when users visit Adobe web applications (like Acrobat, Express, Admin Console, etc.) and reports usage to your SubTracker account. The data helps you track web-based Adobe activity alongside desktop application usage.

Tracked Sites: adobe.com, acrobat.adobe.com, express.adobe.com, adminconsole.adobe.com, and more.
